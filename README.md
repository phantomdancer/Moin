# Moin (墨影截图)

一款 macOS 截图工具，让截图和标注变得简单高效。

体积小巧，仅几MB。

## 下载地址

https://github.com/phantomdancer/Moin/releases

## ✨ 主要功能

### 📸 截图功能
- **区域截图**：自由选择任意区域进行截图
- **多显示器支持**：完美支持多屏幕环境
- **全局快捷键**：默认 `Ctrl + Cmd + A` 快速截图
- **全局快捷键**：默认 `Ctrl + Cmd + O` OCR识别
- **全局快捷键**：默认 `Ctrl + Cmd + W` 录制视频

## 🚀 快速开始

### 安装
1. 下载最新版本的 Moin
2. 将应用拖拽到 Applications 文件夹
3. 首次启动时授予必要的系统权限

## 📋 系统要求

- **操作系统**：macOS 14.0 或更高版本
